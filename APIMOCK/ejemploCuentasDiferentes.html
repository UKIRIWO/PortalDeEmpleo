¿Cómo usar múltiples sesiones?

1. Inicia sesión con una cuenta en esta pestaña

2. Abre una nueva pestaña con la misma URL

3. Inicia sesión con una cuenta diferente

4. Ambas sesiones permanecerán activas independientemente

Explicación Técnica

Este sistema usa sessionStorage para credenciales y localStorage para tokens, permitiendo:

  • Sesiones independientes por pestaña
  • Persistencia de tokens entre pestañas
  • Diferentes cuentas en diferentes pestañas
